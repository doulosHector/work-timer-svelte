# Work Timer

## Development

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Or start the server and open the app in a new browser tab:

```bash
npm run dev -- --open
```

## Build

To create a production version of the app:

```bash
npm run build
```

You can preview the production build with

```bash
npm run preview
```
